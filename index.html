โปรเจค

  • เว็บไซต์ห้องคอมพิวเตอร์ ปัจจุบัน

    โปรเจคนี้เป็นโปรเจคที่ร่วมกันทำกับอีกคนนึง โดยเว็บไซต์นี้เป็นการลงทะเบียนกิจกรรมต่างๆ เช่นการแข่งขัน หรือกิจกรรมภายในโรงเรียน โดยระบบในเว็บไซต์เป็นแบบ one-click เพื่อให้สะดวกในการใช้งาน โดยระบบสร้างจาก PHP มี CSS, Javascript ในการตกแต่งให้ดูสวยงามและระบบภายในยังเป็น Object Oriented Programming : OOP อีกด้วย เพื่อรองรับ Structure ที่เหมาะสมนับว่าเป็นโปรเจคแรกที่ประสบความสำเร็จมากที่สุด โดยโปรเจคอันนี้ได้ใช้ในกิจกรรม กิจกรรม Bebras Challenge 2563 หรือการทดสอบด้านทักษะการคิดเชิงคำนวณ โดยมีนักเรียนสมัครตั้งแต่ระดับชั้น ม.1 จนถึง ม.6 เป็นจำนวน 196 คน ต้องขอบคุณ คุณครูประจำห้องคอมพิวเตอร์ที่ให้โอกาสใช้ระบบนะครับ

    อยากส่องเหรอ กดจิ~
